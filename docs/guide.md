## 适合谁

- 想从零构建 AI 编码代理的开发者
- 想深入理解 Agent 架构设计的工程师
- 想学习大型 TypeScript 项目设计模式的技术人员

## 前置知识

| 技术 | 级别 | 说明 |
|------|------|------|
| TypeScript | 熟悉 | 泛型、类型体操、模块系统 |
| Node.js | 熟悉 | async/await、Stream、子进程 |
| React | 了解 | 组件、hooks、reconciler 概念 |
| LLM API | 了解 | 大模型 API 调用、token、streaming |
| 终端 | 了解 | ANSI 转义码、TTY 基础 |
| Git | 了解 | 分支、工作树、diff |

## 如何使用本教程

1. **顺序阅读**：按 01→13 顺序，每章都建立在前一章的理解之上
2. **选题跳读**：如果你已有 Agent 经验，可直接跳到感兴趣的主题
3. **动手实践**：每篇末尾的"启发与超越"部分给出了实践建议
